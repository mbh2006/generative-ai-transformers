# Transformer Implementation and Attention Analysis

This repository contains the implementation and analysis of **Transformer** models, ranging from building a decoder-only language model from scratch to visualizing attention mechanisms.

## 🚀 Overview

This project is divided into three core tasks:
- **Task 1: Transformer from Scratch:** Implementation of a decoder-only language model using PyTorch. Covers embedding, self-attention, feedforward networks, and residual connections.
- **Task 2: Understanding Attention:** Visualization and interpretation of attention matrices from a pretrained BERT model.
- **Task 3: Pretrained Applications:** Exploration of practical Transformer usage and reflections on performance.
